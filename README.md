# Calculadora de Interés Simple

Este proyecto es una herramienta sencilla diseñada para calcular el interés simple generado por un capital inicial invertido o prestado durante un período de tiempo determinado, aplicando una tasa de interés específica.

---

## 📋 Descripción del Proyecto

La calculadora permite a los usuarios proyectar sus ganancias o costos financieros de manera rápida y clara. Es ideal para entender los conceptos básicos de las finanzas personales y las matemáticas financieras antes de avanzar a modelos más complejos como el interés compuesto.

---

## 🧮 Concepto Matemático

El interés simple se calcula utilizando la siguiente fórmula matemática:

$$I = P \cdot r \cdot t$$

Donde:
* **$I$ (Interés):** El monto total de interés generado.
* **$P$ (Principal / Capital inicial):** La cantidad de dinero invertida o prestada originalmente.
* **$r$ (Tasa de interés):** El porcentaje de interés anual expresado en forma decimal (por ejemplo, 5% se introduce como 0.05).
* **$t$ (Tiempo):** El período de tiempo durante el cual el dinero se presta o invierte, generalmente expresado en años.

### Monto Total
Para calcular el valor total final (Capital + Interés), se utiliza la fórmula:

$$A = P (1 + r \cdot t)$$

---

## 🚀 Características

* **Cálculo instantáneo:** Obtén el interés generado y el monto total acumulado con solo ingresar tres datos.
* **Interfaz amigable:** Diseñada para ser intuitiva y fácil de usar por cualquier persona.
* **Educativo:** Incluye explicaciones de las variables para que el usuario entienda qué se está calculando.
